# Proyecto de Ciencia de Datos  
## Análisis de empresas de Risaralda

Este proyecto analiza información financiera de las empresas más grandes del departamento de Risaralda utilizando Python.

## Objetivo
Explorar patrones en los ingresos operacionales y su relación con variables financieras como activos, pasivos y patrimonio.

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Análisis realizado
- Limpieza de datos
- Análisis exploratorio
- Visualización de datos
- Modelo predictivo con regresión lineal

## Resultados
Se encontró una relación positiva entre los activos y los ingresos operacionales de las empresas.

## Aplicación
Este tipo de análisis permite apoyar la toma de decisiones empresariales y evaluar el desempeño financiero de las organizaciones.
